# S4Abuse

S4Abuse es una plataforma abierta y gratuita para verificar y reportar estafas, fraudes y señales de riesgo digital.

Permite analizar números telefónicos, enlaces y mensajes sospechosos antes de interactuar con ellos, ayudando a prevenir fraudes en tiempo real.

## ¿Qué podés hacer?

- Verificar si un número, link o mensaje está asociado a una estafa
- Reportar fraudes y colaborar con la comunidad
- Consultar indicadores de riesgo digital reutilizados en campañas

## ¿Por qué es importante?

Las estafas digitales reutilizan estructuras: números, enlaces y cuentas.

S4Abuse permite detectarlas antes de que sea tarde.
